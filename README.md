# product-graph-plugin

Public distribution of the [Product Graph](https://product-graph.app) Claude plugin.
Mirrored from a private source repo — edit there, not here.

```
/plugin marketplace add Elun1/product-graph-plugin
/plugin install product-graph@productos
```

Set `PRODUCT_GRAPH_TOKEN` (mint one in the app under **Admin → Tokens**) before
installing. The plugin brings the skills and the `product-graph` MCP server.
